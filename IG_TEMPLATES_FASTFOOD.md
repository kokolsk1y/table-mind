# IG DM шаблоны под фастфуд / кофейни / шаурму / киоски

> **Версия:** 2026-04-30 (новое позиционирование с киоск-режимом)
> **Использовать для:** лидов из LEADS_FASTFOOD_KALININGRAD.md
> **НЕ использовать для:** старых casual/fine dining лидов (LEADS_KALININGRAD.md — архив)

---

## Принципы

1. **Первая строка персонализирует** — упомяни конкретное место/блюдо/факт. Без неё DM = спам.
2. **Один сильный аргумент**, не три. Лучше "разгрузим очередь" чем "разгрузим очередь + мультиязык + апсейл".
3. **Без "AI"** в первых сообщениях. Это плюшка, не основной продукт.
4. **CTA = простой**: "интересно — отвечу подробнее" / "могу показать демо за 5 минут".
5. **Не более 4-5 строк** в первом DM. Длинные простыни не читают.

## Шаблон A — Кофейни самообслуживания (Coffee Master, Coffee express, Eva, T&W)

**Когда:** локальная сеть с 5+ точками. Главный аргумент — **апсейл и мультиязык для туристов**.

```
Здравствуйте! Вижу [N] точек [название] по городу — впечатляюще для локальной сети.

Делаем софт для киосков самообслуживания: гость выбирает напиток сам, к нему автоматически предлагается десерт/доп. Средний чек растёт на 15-25%, на персонал нагрузка та же.

Тариф для сетей точек со средним чеком до 500₽ — 1 990 ₽/мес за точку, 14 дней бесплатно.

Если интересно — могу показать как это будет выглядеть на ваших чашках за 5 минут.
```

За одну точку — 1 990 ₽/мес, для 27 точек предложу пакет дешевле. Готов сделать демо вашего бренда за день — посмотрите бесплатно, никаких обязательств.

Кому передать предложение — вам напрямую или менеджеру по операциям?
```

## Шаблон B — Шаурма / донер (Интер Кебаб, Senor Doner, локальные)

**Когда:** заведение с очередью у кассы. Главный аргумент — **разгрузка кассы**.

```
Здравствуйте! Был у вас [конкретный адрес] — взял [конкретное блюдо]. Очередь в обед была минут 7, заметил.

Делаем экран самообслуживания: гость выбирает шаурму сам, оплачивает наличкой/QR у кассира — кассиру остаётся только собрать. Очередь сокращается вдвое. Не нанимаете никого.

Цена 1 990 ₽/мес за точку, 14 дней бесплатно. Покажу демо за 5 минут — интересно?
```

## Шаблон C — Пиццерии (Додо, Папаша Беппе, локальные)

**Когда:** меню сложнее (топпинги, размеры). Главный аргумент — **гость сам собирает заказ без ошибок**.

```
Здравствуйте! [Заведение] — был, [конкретное блюдо] отлично.

Делаем софт для самозаказа на планшете: гость собирает пиццу сам (размер, бортик, начинка), без вопросов кассиру и ошибок при передаче. Сразу видит итоговую цену, к пицце автоматом — соус, напиток, доп.

Заказ дублируется кассиру в обычную систему (или в наш дашборд если у вас её нет). Цена от 1 990 ₽/мес за точку.

Покажу как это будет выглядеть на вашем меню — интересно?
```

## Шаблон D — Фуд-корт (Кирин, ВДНХ, Али Бобо, Патиссон Markt)

**Когда:** заведение в ТРЦ с большим потоком, разнообразное меню. Главный аргумент — **мультиязык + апсейл**.

```
Здравствуйте! [Заведение] на [фуд-корт ТРЦ] — отличная локация и поток.

Делаем планшет самообслуживания для фуд-кортов: гость выбирает блюдо на любом из 12 языков (туристы из Польши/Германии/Литвы — частые гости в Калининграде), к каждому блюду автомат предлагает напиток/закуску. Средний чек растёт на 20%.

Цена 5 900 ₽/мес для меню со сложным выбором, 14 дней бесплатно. Покажу демо вашего меню — интересно?
```

Здравствуйте! Прочитал что Европа открывает фуд-холл на 3-м этаже в конце 2026 — большая новость для города.

Делаем системное решение для фуд-кортов: единые планшеты самообслуживания для всех арендаторов с общим UX, мультиязык 12 языков (для туристического потока Калининграда). Каждый арендатор получает свой брендинг и меню, но общий стандарт обслуживания фуд-холла.

Можем обсудить как это интегрируется в проект фуд-холла — это даст вам сильный selling point для арендаторов: "у нас современный фуд-холл уровня московских ТРЦ".

Кому передать материалы — вам напрямую или директору по развитию?
```

## Что отвечать на типичные возражения

**"У нас уже есть iiko/Poster"**
→ "Отлично, мы дополняем, а не заменяем. Заказ с нашего киоска уходит сразу к вам в [iiko/Poster] — кассир видит готовый чек. Если хотите — можем интегрировать в Phase 2 (тариф Сеть)."

**"Дорого"**
→ "Какой бюджет рассматриваете? У нас есть тариф Лавка 1 990 ₽/мес — это меньше зарплаты одного дня кассира. Выбираем под вашу боль, не общий пакет."

**"Не уверены что наши гости будут пользоваться"**
→ "Можем поставить рядом с обычной кассой как опцию — кому удобнее планшет, тот выберет. По нашим данным [когда они будут — пока их нет] около 40% выбирают самостоятельный заказ. И 14 дней trial бесплатно — попробуете без риска."

**"Нужно посоветоваться/подумать"**
→ "Конечно. Давайте я пришлю вам материалы и пример демо для [их заведения] завтра — посмотрите спокойно, без обязательств. Какой у вас удобный канал — этот же DM или email?"

## Что НЕ писать в DM

- ❌ "AI-официант / искусственный интеллект" — отпугивает фастфуд-владельцев
- ❌ "Революционное решение" / "trends 2026" / "цифровая трансформация" — звучит как агентство
- ❌ "Наша платформа использует Claude API и SvelteKit" — никому неинтересно как мы внутри
- ❌ Длинные тексты больше 5 строк в первом контакте
- ❌ "Здравствуйте, я делаю продукт для ресторанов" — без персонализации

## Расписание и отслеживание

- **Не больше 5 DM в день** — IG банит за массовую рассылку
- **Не позже 20:00 МСК** — личные DM в позднее время раздражают
- **Через 3-5 дней — фолоу-ап**: "Здравствуйте, проверяю не потерялось ли сообщение. Если неактуально — просто скажите, не буду беспокоить."
- **Через 10 дней без ответа** — закрыть статус ⏸ и идти дальше
- **Записывать каждый ответ** в LEADS_FASTFOOD_KALININGRAD.md в столбце Статус
