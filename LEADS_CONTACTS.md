## Порядок работы (за один заход, 1-2 часа)

1. Открой [2ГИС Калининград](https://2gis.ru/kaliningrad/search/Рестораны) в отдельной вкладке
2. Для каждой позиции из списка выше:
   - Вбей название в 2ГИС → копируй телефон, часы, сайт
   - Открой сайт → найди Instagram / WhatsApp
   - Если есть Instagram — зайди, найди есть ли контакт «по рекламе/партнёрству» в био
3. Заполни 4 поля минимум: **Телефон, Instagram, Сайт, Контактное лицо** (если видно)
4. Отметь статус ⬜ → 📩 после первого касания
5. Скрипт касания — смотри [OUTREACH.md](OUTREACH.md)
