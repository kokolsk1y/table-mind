## Objective
Создать `static/catalog.json` — демо-меню вымышленного ресторана «Янтарный берег» (Калининград). Файл содержит 45-50 блюд в 14 категориях с реалистичными описаниями (2-3 предложения каждое), ценами, аллергенами, тегами и ссылками на фото. Это ключевой артефакт для продажного демо — качество контента критично.

## Context
Executor должен читать перед началом:
- `c:\Users\ikoko\Projects\table-mind\docs\phases\phase-2\CONTEXT.md` — schema, 14 категорий, правила allergens
- `c:\Users\ikoko\Projects\table-mind\docs\REQUIREMENTS.md` — MENU-01, AI-06, UI-03

Схема каждого элемента:
```json
{
  "id": "string (kebab-case, уникальный)",
  "name": "string",
  "category": "string (одна из 14 категорий)",
  "description": "string (2-3 предложения, развёрнутое для AI-контекста)",
  "price": "number (в рублях, целое)",
  "photo": "string (photos/{id}.webp)",
  "tags": ["string"],
  "allergens": ["string"],
  "weight": "string (например '350 г' или '200 мл')",
  "spicy": "boolean",
  "vegetarian": "boolean"
}
```

Правила:
- `allergens` — обязательное поле. `[]` означает «неизвестно», НЕ «нет аллергенов». Для блюд с очевидными аллергенами (молоко в сырниках, глютен в пасте, рыба в ухе) — указать реалистично. Для блюд где неясно — оставить `[]`.
- `description` — 2-3 предложения, написаны «вкусно» (как в реальном меню), но с конкретикой (ингредиенты, способ приготовления). AI будет использовать эти описания для рекомендаций.
- `tags` — из набора: `"веган"`, `"вегетарианское"`, `"без глютена"`, `"острое"`, `"детское"`, `"хит"`, `"новинка"`, `"авторское"`, `"сезонное"`, `"локальное"`. Каждое блюдо имеет 1-3 тега.
- `price` — реалистичные цены для ресторана среднего+ класса в Калининграде (2024-2025 уровень).
- `photo` — путь `photos/{id}.webp`. Фото будут скачаны позже; в каталоге фиксируем имена файлов.

14 категорий (строго в таком порядке и написании):
1. Холодные закуски (3-4 блюда)
2. Тёплые закуски (3-4 блюда)
3. Салаты (3-4 блюда)
4. Супы (3-4 блюда)
5. Рыба и морепродукты (4-5 блюд)
6. Стейки и мясо (4-5 блюд)
7. Паста и ризотто (3-4 блюда)
8. Гарниры (3-4 блюда)
9. Детское меню (3 блюда)
10. Десерты (3-4 блюда)
11. Безалкогольные напитки (3-4 позиции)
12. Авторские коктейли (3 позиции)
13. Вино (3 позиции)
14. Пиво и сидр (3 позиции)

Калининградский колорит: строганина из палтуса, балтийская корюшка, клопсы, копчёный угорь, марципановый десерт, крафтовое пиво с янтарной темой, сезонная балтийская рыба. Но также универсальная классика: Цезарь, стейк рибай, тирамису, том-ям.
