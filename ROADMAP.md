# Roadmap TableMind

## Фаза 1. MVP (дни 1-3)
- [ ] Шаблон PWA: HTML/CSS/JS, адаптивная вёрстка
- [ ] Подключение Google Sheets API (чтение меню)
- [ ] AI-виджет: чат с OpenRouter (Claude Haiku)
- [ ] Telegram-бот: уведомление «гость зовёт» с номером стола
- [ ] Генератор QR-кодов (с параметром стола)
- [ ] 3 темы оформления (light, dark, coffee)
- [ ] Деплой на Cloudflare Pages
- [ ] README для клиента: как обновлять меню

## Фаза 2. Первый клиент (дни 4-8)
- [ ] Демо на тестовом ресторане
- [ ] Найти первого клиента (бесплатно/5 000 ₽)
- [ ] Установить, обучить администратора
- [ ] Начать сбор метрик

## Фаза 3. Рассылка (дни 9-15)
- [ ] База 200+ заведений Калининграда
- [ ] 10 персональных демо вручную
- [ ] Отправка первых 10 сообщений
- [ ] Анализ ответов, корректировка текста

## Фаза 4. Автоматизация (дни 16-25)
- [ ] n8n workflow: парсинг меню с сайта → Sheets → демо-ссылка
- [ ] Вторая волна: 50 автоматических демо
- [ ] Массовая рассылка с персональными ссылками

## Фаза 5. Кейс и масштаб (дни 26-30)
- [ ] Видео-кейс с первым клиентом
- [ ] Публикация на vc.ru / Telegram
- [ ] Ретроспектива: цифры, выводы, корректировка

## Фаза 6. Pro-пакет (месяц 2)
- [ ] Коннектор Poster POS API
- [ ] Коннектор Quick Resto API
- [ ] Аналитика: дашборд вопросов и просмотров
- [ ] Автоскрытие блюд «нет в наличии»

## Фаза 7. Enterprise-пакет (месяц 3)
- [ ] Коннектор iikoCloud API
- [ ] Раздельные меню (зал/бар/терраса)
- [ ] Кастомизация дизайна под бренд
- [ ] Расширение на другие города
